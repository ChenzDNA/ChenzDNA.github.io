目前计算机本科在读，学了一些框架，做过一些项目，封装了一些工具，差不多就这样吧。

主要使用 Java，Python，JavaScript。对于 Java，我不敢说精通。这门语言发展了这么多年，底层的‘地基’打得很深，jvm 的各种特性看得我头晕；上层的‘建筑’也令人望而生却，微服务、分布式的那一套框架十分庞大，只能一点一点啃。

别看我仓库里面有一些 C++ 的仓库，实际上我才半只脚踏进 C++ 的大门。现在 C++ 水平也就刷个题目，而且 STL 库都不怎么会。我小时候就想做一个游戏开发，创造属于我的世界。但是进了大学视野变宽后，发现“做游戏死路一条”（开个玩笑😜），发现做前端、后端也是创造属于自己的东西，还挺好玩的，也就一直没碰游戏引擎。直到去年暑假，简单上手了 Unity 游戏引擎，发现这又是一座大山，而且我还没有图形相关的知识，只能说先放放吧。

前几天看完了 Vue3 和 TypeScript 的文档，还没正式上手做东西。后端方面我对 Go 语言有点兴趣，一直打算学来着，先把 Java 那一套搞好再说吧。

bye 🖐
